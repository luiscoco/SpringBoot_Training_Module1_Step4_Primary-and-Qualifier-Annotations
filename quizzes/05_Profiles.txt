What is the main purpose of Spring Profiles?
a) To create static configurations
b) To enable conditional bean registration based on the active profile
c) To eliminate the need for configuration files
d) To enforce a single configuration for all environments
Answer: b) To enable conditional bean registration based on the active profile
Explanation: Spring Profiles allow for different configurations based on the active profile, facilitating dynamic behavior in the application. The other options do not accurately reflect the purpose of profiles.
